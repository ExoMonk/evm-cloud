## Quick Install

:::code-group

```bash [macOS (Homebrew)]
brew install ExoMonk/tap/evm-cloud
```

```bash [macOS / Linux (curl)]
curl -fsSL https://github.com/ExoMonk/evm-cloud/releases/latest/download/install.sh | bash
```

```bash [From Source]
git clone https://github.com/ExoMonk/evm-cloud.git
cd evm-cloud && cargo install --path cli
```

:::

## Verify

```bash
evm-cloud --version
```

## CI (GitHub Actions, pinned version)

```yaml
- name: Install evm-cloud (pinned)
  run: curl -fsSL https://github.com/ExoMonk/evm-cloud/releases/download/0.0.1-alpha8/install.sh | bash -s -- 0.0.1-alpha8

- name: Verify evm-cloud version
  run: evm-cloud --version
```

## Prerequisites

The CLI wraps OpenTofu (Terraform-compatible) — you'll also need:

| Tool | Version | Install |
|------|---------|---------|
| **OpenTofu** | >= 1.14.6 | [opentofu.org/docs/intro/install](https://opentofu.org/docs/intro/install/) |
| **AWS CLI** | v2 | [docs.aws.amazon.com/cli](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) |

> **Note:** evm-cloud uses [OpenTofu](https://opentofu.org/), an open-source fork of Terraform. The version constraint `>= 1.14.6` targets OpenTofu versioning (not HashiCorp Terraform). All `terraform` commands in this documentation work with `tofu` as a drop-in replacement.
